# jsdoc-vue

> A jsdoc plugin that parses *.vue files.

# Installation
```shell
$ npm i jsdoc-vue -D
```

# Usage
[Configuring JSDoc with conf.json](http://usejsdoc.org/about-configuring-jsdoc.html)
```json
{
  "plugins": ["node_modules/jsdoc-vue"],
  "source": {
    "includePattern": "\\.(vue|js)$"
  }
}
```

# Test
```shell
npm test
```
